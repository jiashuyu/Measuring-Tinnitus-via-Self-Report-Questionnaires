Reduced vs Full: When doing CFA we consider two models. The first is a reduced (sparse) model where \(\Lambda\) is 0/1. The second is a full model where certain \(\lambda_{ij}=0\) but the rest are unrestricted. In both cases we allow for an unrestricted \(\Phi\). Done in Lavaan.

EFA #: We look at the EFA goodness of fit for different number of parameters. Horizontal lines for legacy subscales. Purpose Compared to the legacy subscales, how much better performance could be achieved with an EFA model? Since EFA is more flexible we expect it to fit better, but if it performs MUCH better this indicates the legacy subscale may be poor.
